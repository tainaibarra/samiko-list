# Samiko List 🐶💜

##  Descripción
Samiko List es una aplicación web desarrollada con React que permite gestionar una lista de tareas.  
El usuario puede agregar tareas, visualizarlas en pantalla y eliminarlas cuando lo desee.  
Además, la aplicación incluye un modo claro y oscuro para mejorar la experiencia visual.

---

##  Objetivo del proyecto
El objetivo de esta aplicación es aplicar los conceptos básicos de React, incluyendo:
- Uso de componentes
- Manejo de estado con useState
- Interacción con el usuario
- Integración de HTML, CSS y JavaScript

---

##  Tecnologías utilizadas
- React
- Vite
- JavaScript
- HTML
- CSS

---

##  Componentes principales

### App.jsx
Componente principal que:
- Maneja el estado de las tareas
- Controla el input
- Controla el modo claro/oscuro
- Renderiza la lista de tareas

### TodoItem.jsx
Componente encargado de:
- Mostrar cada tarea individual
- Permitir eliminar una tarea

---

##  Flujo de la aplicación
1. El usuario escribe una tarea en el input.
2. Al presionar el botón "Agregar", la tarea se guarda en el estado.
3. La lista se actualiza automáticamente en pantalla.
4. El usuario puede eliminar una tarea con el botón .
5. Puede cambiar entre modo claro y oscuro con el botón correspondiente.
